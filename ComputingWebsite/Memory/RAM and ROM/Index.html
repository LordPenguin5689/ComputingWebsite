  • RAM and ROM
  • Virtual Memory

    RAM:

  • RAM stands for Random Access Memory.
  • It hold all programs / instructions / data that is being run at that time (unless there is no room left).
  • It is volatile, meaning it loses all of its contents when disconnected from the electricity

    ROM:

  • ROM stands for Read Only Memory.
  • It holds data held permanently in memory, for example the BIOS.
  • Embedded System can hold their software on the ROM as the software never needs to be edited.

    Virtual Memory:

  • Virtual Memory is used when the RAM runs out of room to hold data.
  • In this case a portion of the Hard Drive is partitioned off and used as RAM.
  • When the RAM is filled, the program last used will be moved to virtual memory.
  • When you start using that program again, it is moved back into the RAM and another program is swapped out, meaning it takes longer to open.
