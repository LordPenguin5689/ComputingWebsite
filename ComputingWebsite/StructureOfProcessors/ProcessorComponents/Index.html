Colour code: red – fetch, green – decode, blue – execute

    Assembly Code Instructions:
  • INP - Input
  • STA x - Stores number in accumulator to address x
  • LDA x - Load the number in address x into the accumulator
  • OUT - Output the number in the accumulator
  • ADD x - Add the number at address x to the accumulator
  • SUB x - Subtract the number at address x to the accumulator.
  • HLT - End the program
