    Factors that affect the performance of a CPU, (in rough order of biggest impact to smallest).

  • Speed of the clock. Faster clock speed means more of the entire fetch-execute cycles can be performed.
  • Speed of the Data Bus. Faster data transfer allows for faster fetch part of fetch-execute cycle.
  • Width of the Data Bus. The amount of data that can be transfered at once is very important in fetching data.
  • Number of Cores. More cores means more simultaneous fetch-execute cycles being performed at once. Also each core should have a little cache.
  • Size of the registers. Directly affects width of data bus.
  • Size of the Cache. Bigger cache means more recent application can be stored, meaning less time taken to fetch data for recently use applications.
  • Size of the RAM. Bigger RAM means more programs can be run at the same time.
